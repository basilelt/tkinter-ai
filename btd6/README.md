# BTD6 Clone

This is a simplified clone of Bloons Tower Defense 6 built with Tkinter.

## Files

- main.py: Main game file containing all classes and game logic

## Features

- Multiple tower types (Dart, Tack, Sniper)
- Different bloon types (Red, Blue, Green with increasing health/speed)
- Wave system with increasing difficulty
- Money system for purchasing towers
- Health system (lose when bloons reach end)
- Basic projectile system and collision detection

## How to Run

Run `python main.py` in the test directory.
