# Zelda Clone

A simple clone of The Legend of Zelda (NES) built with Python and Tkinter.

## Features

- Top-down 2D gameplay
- Player movement with WASD or arrow keys
- Basic enemy AI
- Item collection system
- Dungeon exploration
- Combat mechanics

## How to Run

1. Ensure Python 3.x is installed
2. Install requirements: `pip install -r requirements.txt`
3. Run the game: `python main.py`

## Controls

- WASD or Arrow Keys: Move
- Space: Attack

## Game Architecture

- `Game`: Main game controller and loop
- `Player`: Player character with movement and combat
- `World`: Tile-based world representation
- `Enemy`: Enemy entities with AI
- `Item`: Collectible items
- `Inventory`: Player's item storage
- `Dungeon`: Dungeon system with rooms

## Development

This game is built as a learning project to demonstrate game development concepts using Tkinter.
